# Zone Builder

---

## Installation

OS: Verified running on Windows 7 with Node and NPM.

### Running the server & client
1. Clone repo: https://github.com/kustomzone/kzone.git
1. `cd kzone`
1. `run_server`
1. `run_client`

Open `localhost:9000/?connect=localhost:9001/test.xml` in your browser.
